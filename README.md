# n8n-resources
A repo for useful n8n docs, links, workflows, etc. for our n8n community of practice.

## What belongs here
- Reusable and shareable n8n workflows (JSON)
- Docs about integrations, auth, APIs, gotchas
- Links to good docs, blog posts, tools
- Example patterns we’ve discussed in meetups

## What does NOT belong here
- Client-specific or sensitive workflows
- Secrets, tokens, credentials
- Half-finished experiments with no context

## How to contribute
- Add files directly to the relevant folder
- Suggest new sub-folders if that will make everything clearer
- Use clear self-documenting filenames
- If you’re unsure, open a Pull Request
